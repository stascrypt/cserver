перед установкой откройте нужные порты:
sudo ufw allow 6789
sudo ufw allow 27015/udp
Для запуска сборки образа выполните: 
docker compose up -d 
После запуска выполните:
./install эта команда запустит установку 
сервера кс 1.6 ReHLDS + Metamod-r + Reunion + AMXX + ReGameDLL...
если выдаст ошибку просто запустите установку снова
команды для управления сервером: 
./start       – запустить
./stop        – остановить
./restart     – перезапустить
./console     – открыть консоль
